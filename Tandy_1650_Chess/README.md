# Tandy 1650 Computerised Portable Sensory Chess

The Tandy 1650 uses the SciSys TE1 (0609V171) which implements the [LogiChess program](https://www.chessprogramming.org/LogiChess) in an Hitachi HD6301V microcontroller.<br>

![Tandy 1650](/Tandy_1650_Chess/Tandy_1650_Chess.png)

## Chess Co-Processing Unit
So, I've long wondered whether I could shoehorn the SciSys TE1 into an 8-bit computer so that it could be used as a chess co-processor.<br>
